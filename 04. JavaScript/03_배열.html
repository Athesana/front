배열

자바스크립트에서 배열은 자료형 지정하지 않기 때문에 하나의 배열 안에 모든 자료형의 데이터 보관이 가능하다.

배열 선언 및 초기화

배열 선언 시 크기를 지정하지 않고 선언할 수도 있고 크기를 정하고 선언할 수도 있다.

배열의 메소드

배열도 하나의 객체이므로 배열에서 활용할 수 있는 메소드들이 존재한다.

indexOf()

배열에서 요소가 위치한 인덱스를 반환한다.
없는 값을 넣으면 '-1' 반환
indexof()는 배열에 같은 요소가 여러 개 있으면 앞쪽에서부터 가장 처음 만난 인덱스의 값을 반환한다. 

            cf. includes() : 배열 안에 요소를 포함하고 있는지 true, false로 리턴
                lastIndexOf() : 배열에 같은 요소가 여러 개 있으면 맨 마지막에 있는 요소를 반환한다.

concat()

여러 개의 배열을 결합하여 새로운 배열을 리턴한다.

join()

배열의 요소들을 결합하여 문자열로 반환한다.

reverse()

원본 배열의 순서를 뒤집는 메소드이다.

sort()

원본 배열을 오름차순(문자를 기준으로)으로 정렬하는 메소드이다.

push(), pop()

push() : 배열의 맨 뒤에 값(요소)를 추가한다.
pop() : 배열의 맨 뒤에 값(요소)를 반환 후 제거한다.

shift(), unshift()

shift() : 배열의 맨 앞의 요소를 반환 후 제거한다.
unshift() : 배열의 맨 앞에 요소를 추가한다.

slice(), splice()

slice() : 배열의 요소를 뽑아내는 메소드이다. (begin, end(end미포함))
splice() : 배열의 인덱스 위치에 요소 제거 및 추가하는 메소드이다. (star index, delete count, [item])

            - splice() : 배열의 원하는 특정한 부분을 리턴하는 것, 새로운 배열을 만들어서 반환하기 때문에 원본 배열에는 영향을 주지 않는다.
            - splice(idnex) : 인덱스 하나만 지정하고, 몇 개 삭제할건지 정하지 않으면, 지정 인덱스부터 뒤에 있는 모든 배열 삭제
            - splice(star index, delte count, item) : 삭제한 자리에 새로운 요소를 순서대로 추가하면서 기존 요소들은 인덱스가 뒤로 밀린다.
            - splice는 삭제된 배열의 요소가 결과 값으로 반환되고, array 자체의 값은 바뀐다. = 원본 배열에 영향을 미친다. (새로운 배열을 반환하는 것이 아니다.)

split()

split(seperator: string|RegExp [, limit: number]) 스트링을 전달된 Seperator(String이거나 정규표현식)을 가지고 여러 가지 문자열로 잘게 나누어 준다.
seperator를 지정하지 않으면 문자열 자체가 통채로 배열 0번째 index에 들어가게 된다. 그래서 구분자를 써줘야 한다.
limit : 리턴 받을 배열의 사이즈를 지정할 수도 있다. (옵션 사항)

find()

find(predicate: (this: void, value: T, index: number, obj: T[]), thisArg) : 콜백함수를 만들어서 매개변수로 전달한다.
전달한 콜백함수(화살표 함수 가능)는 배열 안에 있는 모든 요소들을 차례대로 호출하는데 전달된 콜백함수가 true이면, 만약 찾지 못하면 undifined을 리턴
true를 리턴하면 find() 함수는 즉시 종료되고 첫번째로 찾아진 요소를 리턴한다. 즉, 리턴 값은 Boolean의 형태이다.

filter()

콜백함수를 전달하고 true인 요소만 가지고 새로운 배열을 만들어서 전달하는 함수이다.

map()

배열 안에 들어있는 요소 하나하나를 다른 요소로 변환해준다. 전달한 콜백함수가 어떤 역할을 하냐에 따라서 mapping 새롭게 가공 되어서 새로운 값으로 바뀐다.
배열 안에 있는 요소들을 우리가 원하는 함수를 이용해서 다른 방식의 데이터로 만들고 싶을 때 사용한다.
콜백 함수에서 한 줄로 작성하지 않고 길어질 때는 인자는 최대한 이해하기 쉽게 사용하는 것이 좋다.

some()

배열의 요소 중에서 콜백 함수가 리턴이 true가 만족되는 값이 하나라도 있는지 없는지를 확인하는 함수이다.
cf. every() 배열 안의 모든 값이 작성한 조건을 만족하는지 true, false로 리턴한다.

reduce()

콜백함수 또는 이니셜 value(초기값)를 전달해서 배열 안에 있는 모든 요소마다 호출이 되고,
콜백 함수에서 리턴되는 값은 함께 누적된 결과 값을 리턴한다. 배열에 있는 모든 요소들의 값을 누적하고 함께 모을 때 사용하는 함수이다.
