jQuery 개요

jQuery 란?

기존에 자바스크립트 만으로 구현하기 복잡했던 구문 작성들을 간소화하기 위해 '존 레식' 이라는 사람에 의해 개발된 자바스크립트 라이브러리이다.

jQuery의 장점

  1. DOM과 관련된 처리를 쉽게 구현할 수 있다.
  2. ajax 통신, 이벤트 처리 등을 폭넓게 지원한다.
  3. 별도의 플러그인을 통해 차트, 슬라이드 쇼, 테이블 등을 간단히 구현할 수 있다.

jQuery 라이브러리 연결 방법

script 태그를 사용해서 jQuery 관련 .js 파일을 가져온다.

1. jQuery 라이브러리 다운로드를 통한 연결 (오프라인)
- js 파일을 직접 다운로드해서 저장한 후 저장된 파일 경로를 지정하는 방법

2. CDN(Content Delivery Network)을 통한 연결 (온라인)
- 라이브러리를 제공하는 사이트의 주소를 이용하여 파일 경로를 지정하는 방법

jQuery 시작하기

1. jQuery는 문서에서 DOM 요소가 다 로드된 후에 실행되어야 한다.
- 자바스크립트에서 해당 내용은 "window.onload = function(){실행내용};" 과 같이 작성한다.
해당 문서에서 한 번만 작성이 가능하다. 여러 번 작성할 경우(복수 개 설정)는 가장 마지막에 작성된 내용이 적용된다.
- jQuery에서는 "$(document객체).ready(function() {실행 내용});" 과 같이 작성한다.
해당 문서에서 여러 번 작성 가능하다. (but, 한 번만 사용하는 것을 지향한다.)

2. jQuery에서의 onload를 의미하는 구문(3가지 방법)
- $(document객체).ready(function() {실행 내용});
- jQuery(document).ready(function() {실행 내용});
- $(function() {실행 내용});

3. jQuery 기본 작성 형태
- $('선택자').메소드('속성명', '속성값');
